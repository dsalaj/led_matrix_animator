## // JACK IN

A **zero-dependency, single-file** animation studio for LED matrices. No server. No build step. No cloud. Open `index.html` in a browser and start pushing photons.

Built for the kind of hardware that glows in dark server rooms ‚ÄĒ P5 pixel panels, P10 outdoor matrices, custom RGB grids. Exports looping GIFs. Shares state via URL. Runs entirely in your browser.

## // FEATURES

### Matrix Resolution
Set any custom width √ó height for your LED grid. Landscape for marquees, portrait for tower panels. Orientation-sensitive controls auto-adapt when you switch modes.

### LED Pitch Modes
Simulate real-world hardware ‚ÄĒ from tight indoor displays to wide-pitch outdoor panels:

|------|-------|-------------------|

### Animation Layers
Stack layers like a deck of ICE. Each is composited in order, top to bottom.

**MATRIX RAIN** ‚ÄĒ Cascading character rain with independent head/tail color control. Katakana, Latin, or mixed charset. Adjustable density, speed, and trail length. The classic.

**TEXT** ‚ÄĒ Scrolling, bouncing, pulsing, static, or typewriter text. Horizontal or vertical orientation with independent direction control (‚Üź/‚Üí/‚ÜĎ/‚Üď). Per-layer glitch probability. Works with VT323, Share Tech Mono, or system monospace. Supports full Unicode ‚ÄĒ katakana welcome.

**GLITCH** ‚ÄĒ Corrupted display artifacts: block corruption, scanline tears, chromatic aberration, or raw noise. Dial the intensity from subtle interference to full signal death.

**NOISE** ‚ÄĒ Pixel-level phosphor noise. Adds analog warmth to any composition. Low intensity is invisible; high intensity looks like a dying CRT in the rain.

## // ANIMATION MATH

All animations are deterministic functions of `t ‚ąą [0, 1)`. This guarantees perfect GIF loops ‚ÄĒ the last frame always flows back into the first.

Scroll, bounce, and pulse speeds are internally snapped to integers so the modulo condition `f(0) = f(1)` is always satisfied. Matrix Rain column speeds are also integer-snapped with per-column variation for organic feel without loop drift.
